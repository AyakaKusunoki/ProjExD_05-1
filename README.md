# ゲーム のタイトル
食いしん坊こうかとん（仮）
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
上から落ちてくるアイテムをキャッチしてキャラクターを満腹にするゲーム

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス
### 担当追加機能
*視界不良(担当:宮野瑛奈):毒キノコをキャッチすると5秒間視界が白く見えづらくなる機能
